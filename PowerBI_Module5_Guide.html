Module 4

Data Cleansing & Transformation

Master the Power Query Editor to shape, clean, and optimize your data before building visualizations.

Training Progress 0/14
01

Entering Power Query

Click Transform Data on the Home ribbon to open the Power Query Editor.
Identify the Queries Pane (left), Data Preview (center), and Applied Steps (right).
💡
Every change you make is recorded in 'Applied Steps'. You can delete or re-order them at any time.
02

Column Hygiene

Select multiple columns using Ctrl and click Remove Other Columns to reduce data noise.
Double-click column headers to rename them into user-friendly titles (e.g., "Cust_ID" to "Customer ID").
Ensure every column has the correct Data Type (Text, Decimal Number, Date, etc.) via the icon next to the header.
03

Handling Nulls & Errors

Use Replace Values to fill nulls with "0" or "Unknown".
Right-click a column and select Remove Errors if you have unfixable data artifacts.
UI Preview: Applied Steps Pane
04

Text Transformations

Use Split Column by Delimiter (e.g., split "City, State" by the comma).
Apply Format > Trim to remove trailing whitespace from imported text.
Apply Format > Capitalize Each Word for consistent naming conventions.
05

Advanced Transformations

Unpivot Columns: Convert attribute columns into a single attribute/value pair for easier analysis.
Conditional Column: Use 'Add Column' to create logic like if [Sales] > 1000 then "High" else "Low".
Merge Queries: Join two tables (like a VLOOKUP) based on a matching ID column.
06

Loading the Data

Click Close & Apply to save your changes and load the clean data into the Model.
